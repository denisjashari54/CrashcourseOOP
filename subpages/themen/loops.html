Loops

Was sind Loops?

Die deutsche Übersetzung für Loop ist Schleife. Was macht denn eine Schleife? Sie geht rings herum. Im Programmieren ist das ähnlich, wenn man eine Loop als Prozessablaufplan darstellt, sieht sie sehr ähnlich aus, wie eine Schleife in einer Schnur.

Im Code könnte es dann so ähnlich aussehen. (Hier in C#)

Man unterscheidet hauptsächlich zwischen zwei Arten von Loops, die While-Loop und die For-Loop.

Die While-Loop war schon im obigen Beispiel zu erkennen. Nun schauen wir uns die For-Loop ein wenig genauer an.

Wenn man die beiden vergleicht, erkennt man, dass die While-Loop sich wiederholt, bis das Argument falsch ist.

Im Gegensatz zu dieser wiederholt sich die For-Loop nur so oft, wie noch vor dem Ausführen der Schleife angegeben.

Was kann man mit Loops machen?

Mit Loops kann man recht vieles machen. Fast alle Programme enthalten Loops, da sie einfach so praktisch sind. Man kann bestimmen, wie oft oder bis wann sie sich wiederholen sollen, dies nimmt dem Menschen sehr viel Arbeit ab.

Zum Beispiel sind Uhren praktisch nur Loops. Es werden pro 60-mal Minuten Hochzählen, je eine Stunde hochgezählt. Das Ganze erstreckt sich dann noch weiter in die Sekunden und so weiter.

Eine Failsafe-Funktion bei einem User-Input kann auch in Form von einer Schleife programmiert werden. Wenn die While-Loop immer überprüft, ob der Input leer ist oder nicht, kann man keinen leeren Input geben. Der Inputprozess würde sich nochmal wiederholen.
